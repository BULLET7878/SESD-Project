# BluMart Client

The frontend application for BluMart, a modern grocery and snack delivery service.

## Tech Stack
- **React**: UI library
- **Vite**: Build tool and dev server
- **TailwindCSS**: Styling
- **React Router**: Frontend routing
- **Axios**: API requests

## Development

```bash
npm install
npm run dev
```

## Features
- Interactive product catalog
- Category-based filtering
- Shopping cart functionality
- Responsive design
