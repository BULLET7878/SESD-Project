# BluMart Server

Backend logic and API for the BluMart grocery application.

## Tech Stack
- Express
- MongoDB
- Mongoose
- JSON Web Token (JWT)
- Stripe Integration
- Cloudinary Integration
- Multer for file uploads

## Environment Variables
Ensure you have the following environment variables set in a `.env` file:
- `MONGODB_URI`
- `STRIPE_SECRET_KEY`
- `CLOUDINARY_CLOUD_NAME`
- `CLOUDINARY_API_KEY`
- `CLOUDINARY_API_SECRET`
- `JWT_SECRET`
- `ADMIN_EMAIL`
- `ADMIN_PASSWORD`
